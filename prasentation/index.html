Wie funktioniert GPS-Tracking?

    GPS steht für Global Positioning System

  • ist ein Satellitennavigationssystem, das die genaue Position auf der Erdoberfläche bestimmt
  • basiert auf einem Netzwerk von Satelliten, die um die Erde kreisen

    • Satelliten senden ständig Signale aus, die von GPS-Empfängern auf der Erde empfangen werden
    • GPS-Empfänger empfängt Signale von mehreren Satelliten gleichzeitig
    • jeder Satellit sendet Informationen über seine Position und die genaue Zeit
      • => GPS-Empfänger verwendet diese Informationen, um die genaue Position zu berechnen

	xr = c1[0][0] - c2[0][0]
	yr = c1[0][1] - c2[0][1]
	r1 = c1[1]
	r2 = c2[1]

	if (xr == 0 and yr == 0):
		return circlesSameSpot(c1, c2, False)
	
	if (math.sqrt(xr**2 + yr**2) > r1 + r2):  # Punkte zu weit entfernt 
		raise("Circles are too far apart") 

	d = (r2**2 - r1**2 - xr**2 - yr**2) / (2 * xr)
	a = 1 + (yr**2) / (xr**2)
	b = -2 * (d * yr) / xr
	c = d**2 - r1**2

	y = useABC(a, b, c, False)

Experimentelles Konzept auf der 3D Ebene

Fast Fourier (FFT)

  • ist ein mathematisches Verfahren, das in der Signalverarbeitung verwendet wird
  • ermöglicht die verschiedenen Frequenzanteile eines Signals zu analysieren

  • das Mikrofon nimmt alle Frequenzen auf, d.h. auch Hintergrundgeräusche
    => FFT zur Trennung von Stör- und Zielfrequenz

Hardware Grenzen

  • der Audio-Treiber startet nach einer unbestimmten Zeit sich neu
    • die Latenz des Mikrofons ändert sich davon abhängig

    => stark abweichende Resultate
